# 首页重新设计 - 设计文档

## 设计目标

将首页从 Bento 卡片网格风格重新设计为极简内容优先的布局，参考 rauno.me 的设计语言，在保持克制美学的同时给摄影和项目更多视觉空间。

## 设计决策

- 风格：极简文艺型，大量留白，精致排版，克制的动画
- 信息层级：内容优先，精简的个人介绍后直接进入内容展示
- 动画：适度点缀（滚动淡入 + hover 效果），去掉粒子特效等重效果
- 参考：rauno.me 的设计语言

## 页面结构

### 1. Hero 区

- 名字：中等大小标题，字重 400-500，居左对齐
- 身份描述：一句话，灰色次要文字
- 社交链接：一行小图标（GitHub、Twitter 等），克制
- 大量上下留白
- 最大宽度 640px 居中
- 去掉：Cover 粒子特效、三行大标题

### 2. 文章区（Writing）

- 区块标题：小号灰色 + "View all →" 链接
- 列表形式：每行一篇，左侧标题 + 右侧日期
- 不显示描述文字
- 最多 5 篇最新文章
- hover：标题颜色加深 + 微妙右移（2-3px）
- 最大宽度 640px

### 3. 摄影区（Photography）

- 区块标题：同上风格
- 布局：2-3 列网格，4-6 张精选照片
- 照片：保持原始比例，统一高度（240-280px）
- 宽度扩展到 960px（比文字区更宽，形成节奏变化）
- hover：微微放大（scale 1.02）+ 显示拍摄地点浮层
- 不显示 EXIF 信息
- 圆角：4-6px

### 4. 项目区（Projects）

- 区块标题：同上风格
- 布局：2 列网格
- 卡片：项目名称 + 一句话描述 + 技术栈标签（小号灰色）
- 可选：项目截图/缩略图
- 卡片样式：细边框（1px），hover 时边框颜色加深
- 宽度：960px，与摄影区一致
- 展示 2-4 个项目

## 设计规范

### 配色

- 保持现有 zinc 灰度体系
- 亮色/暗色双模式
- 不引入新的强调色

### 字体

- 保持现有系统字体栈
- 不加载网络字体

### 间距

- 区块之间：80-120px
- 内部元素：紧凑

### 动画

- 滚动时各区块淡入（opacity + translateY）
- 使用 Framer Motion
- 去掉粒子特效和 Cover 组件

### 响应式

- 移动端：单列
- 摄影区：2 列 → 1 列
- 项目区：2 列 → 1 列

## 移除的元素

- Bento 网格（音乐、位置/时钟、技术栈卡片全部移除）
- Cover 粒子特效组件
- TechDeck 翻转动画
- TimeDisplay 实时时钟

## 保留的元素

- Header 导航栏（不变）
- Footer 页脚（不变）
- 暗色/亮色模式切换
- 国际化（中英双语）

## 技术栈

- Astro 5 + React 19 + Tailwind CSS 4（不变）
- Framer Motion（保留，但减少使用量）
- @remixicon/react（图标库，保留）
